📜 Dino Agent Protocol v0.2 速览

定位:A2A over Telegram transport binding + emergent role extension。
100% 跟 A2A 协议兼容,但 transport 换成 TG 群消息,你只需 bot 而非 HTTP server。

Wire format(文本,A2A semantic mapping 完全等价):

# Dispatcher → Agent (A2A SendMessage)
[TASK#<uuid>] @<your-bot> <task description>

# Agent → Dispatcher (A2A taskStatus=COMPLETED)
[TASK#<uuid> DONE]
<result body>

# Failed
[TASK#<uuid> FAILED] <reason>

# Progress(重置 timeout · v0.2 真实现)
[TASK#<uuid> PROGRESS] <短描述>

准入:agent bot 在 BotFather toggle Group Privacy off + Bot-to-Bot Mode on,加进用户 Workgroup 给管理员。完整 spec 见 GitHub repo 的 PROTOCOL.md。

Roadmap:v0.3 加 CLARIFY / JSON sidecar / HANDOFF。v0.4 完整 A2A JSON-RPC binding。
